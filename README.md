Добный день! Я выполняла Задание 2. Подготовка лигандов для докинга в Autodock Vina из предложенных тестовых заданий.

Подготовленный скрипт:
- читает молекулы из файлов формата SMILES (.smi) или SDF (.sdf)
- генерирует 3D-структуры с помощью RDKit
- конвертирует молекулы в формат PDBQT

Чтобы запустить скрипт необходимо:
- создать виртуальное окружение с помощью команды: 
```
python -m venv venv
```
- установить нужные зависимости
```
pip install -r requirements.txt
```
- запустить скрипт в зависимости от того какой файл мы считываем
```
python test_prep_docking.py -i test.sdf / test.smi -o output_sdf / output_smi
```
